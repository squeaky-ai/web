### Requirements
- Node.js v14

### Installation
```shell
$ git clone git@github.com:squeaky-ai/app.git
$ cd squeaky
$ yarn install
```

### Fetching the latest GraphQL types
```shell
$ yarn codegen
```

### Run the server
```shell
$ yarn dev
```
