### Requirements
- Node.js v16

### Installation
```shell
$ git clone git@github.com:squeaky-ai/web.git
$ cd web
$ yarn install
```

### Fetching the latest GraphQL types
```shell
$ yarn codegen
```

### Run the server
```shell
$ yarn dev
```
